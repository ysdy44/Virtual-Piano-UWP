 Windows 10 app 应用商店地址:

## 功能说明

V 钢琴 是一款基于微软GS波表合成器的 Midi 工具，它可以演奏 128 种乐器、和弦以及打击乐器。


<br/>

## 开发环境

|Key|Value|
|---|---|
|系统要求| Windows 10 秋季创意者更新 或更高|
|开发工具|Visual Studio 2022|
|编程语言|C#|


<br/>

## 部署说明

> 下载整个工程，打开`Virtual Piano.sln`文件，启动工程`Virtual Piano`，试试看是否可以工作。
